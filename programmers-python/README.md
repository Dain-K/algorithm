## 최대공약수 math.gcd

```python
import math

a = math.gcd()
b = math.gcd(0, 2, 4)
c = math.gcd(10, 5, 100)

print(a) # 0
print(b) # 2
print(c) # 5
```

## 최소공배수 math.lcm()

```python
import math

a = math.lcm()
b = math.clm(0, 2, 4)
c = math.lcm(10, 5, 100)

print(a) # 1
print(b) # 0
print(c) # 100

```
